**PWA (Progressive Web App)** = **installable web app** ‚Äî runs like a native app, but is just a website.

---

### What it does:
| Feature | Description |
|--------|-------------|
| **Installable** | Add to home screen (iPhone/Android) ‚Üí opens full-screen, no browser bar |
| **Offline** | Works without internet (after first load) |
| **Push notifications** | Optional (not needed here) |
| **Fast** | Cached locally |

#### 3. `service-worker.js`
```js
const CACHE = 'qv-notary-v1';
const FILES = ['/', '/index.html', '/manifest.json', '/icon-192.png', '/icon-512.png'];

self.addEventListener('install', e => {
  e.waitUntil(caches.open(CACHE).then(cache => cache.addAll(FILES)));
});

self.addEventListener('fetch', e => {
  e.respondWith(caches.match(e.request).then(resp => resp || fetch(e.request)));
});
```

---

#### 4. Register SW in `index.html` (add to `<head>`)
```html
<script>
  if ('serviceWorker' in navigator) {
    window.addEventListener('load', () => {
      navigator.serviceWorker.register('/service-worker.js');
    });
  }
</script>
```
