**Quantum-safe file signing & verification — everywhere.**

## Features

- Sign any file with **NIST-approved Dilithium (ML-DSA)**
- Verify with **public key + JSON proof**
- Works **offline**, no internet, no logs
- Runs on **Tails, phone, desktop, browser, TPM**
- **Zero trust** — math only

## Quick Start

### CLI (Linux/macOS/Windows)
```bash
./FINAL_Notary_CLI_v1.0_std_rust sign document.pdf
# → document.pdf.sig.json
```

### Web / PWA
1. Open `FINAL_Notary_WEB_app_v1.0.html`
2. Drag file → Sign → Download proof

### Android / iOS
Install APK/IPA → Pick file → Sign

### TPM-HSM
```bash
./FINAL_Notary_TPM-HSM_v1.0_core_sign document.pdf
# Key never leaves chip
```
